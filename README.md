## 🌟 Overview
Ek high-end, visual-heavy portfolio design jo **Graphic Designers** aur **Creative Developers** ke liye optimize kiya gaya hai. Isme modern typography, cream-gold color palette, aur interactable widgets ka use hai.

### 🎨 Design Philosophy
* **High-End Aesthetics:** Minimalist cream background with bold black accents.
* **Bento Grid UI:** Information ko chunks mein divide kiya gaya hai taaki user experience smooth rahe.
* **Responsive Architecture:** Mobile aur Desktop dono par layout fix aur clean dikhta hai.

## 🚀 Highlights of the Code

### 1. **Visual Social Widgets** 📱
Har social platform ke liye alag personality di gayi hai:
* **WhatsApp:** Clean grey card with metadata.
* **Instagram:** Vibrant yellow brand color with arrow micro-interaction layout.
* **Email:** Professional dark mode card.

### 2. **Professional Timeline** 🎓
Education aur Experience sections ko **stars (★)** aur **italicized metadata** ke saath structure kiya gaya hai, jo readability badhata hai.

### 3. **Software Competency Grid** 💻
Adobe suite ke icons ko custom CSS boxes mein represent kiya gaya hai jo actual software splash screens ki yaad dilate hain.
